# amropick-vision Python package

This folder will contain the stand-alone python package of the developed AMROPick Vision System.

You should install this package to your local virtual environment with:

```
conda activate amropick
cd amropick-vision
pip install -e .
```

The usage of the package is shown in [../computer-vision-notebooks/amropick-demo.ipynb](../computer-vision-notebooks/amropick-demo.ipynb)

The object detection method is based on an approach with multiple processing stages which is illustrated in [../computer-vision-notebooks/predict-yolov8-multiple-stages.ipynb](../computer-vision-notebooks/predict-yolov8-multiple-stages.ipynb).
